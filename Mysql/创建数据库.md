我们可以在登陆 MySQL 服务后，使用 create 命令创建数据库，语法如下:

CREATE DATABASE 数据库名;

	mysql> create database hello;
	Query OK, 1 row affected (0.02 sec)

	mysql> show databases;
	+--------------------+
	| Database           |
	+--------------------+
	| db                 |
	| hello              |
	| information_schema |
	| mysql              |
	| nodesystem         |
	| performance_schema |
	+--------------------+
	6 rows in set (0.01 sec)

	mysql>
